Example Client Library Setup

A particular library can be requested by category using the cq:includeClientLib tag. This tag produces, among other thins, script and link HTML tags necessary for inclusion of the aggregate JavaScript and CSS libraries.

In our example, we might include a request for the "myapp.main" category. This category embeds the "myapp.jquery", "myapp.bootstrap", and "myapp.components" categories. Thus, an inclusion of the library answering to the "myapp.main" category includes our global libraries and our component-specific libraries.
